# MT-syntax
本仓库提供 [MT管理器](https://mt2.cn/) 软件中的第三方语法高亮文件，为开发者提供更多的非内置语法高亮文件

## 快速开始
访问仓库的 **[mtsx](./mtsx/)** 文件夹下载对应语言的语法文件。

> [!TIP]
> 本仓库提供的语法文件需要 MT管理器 2.16.0 及其以上版本

**下载文件**
1. 选择并点击 .mtsx 文件
2. 点击右侧的 … 按钮
3. 点击 Download 下载文件

**安装文件**
1. 使用 MT 管理器打开 `.mtsx` 后缀的文件
2. 在对话框中点击安装

**卸载文件**
1. 使用 MT 管理器打开 `.mtsx` 后缀的文件
2. 在对话框中点击管理
3. 长按需要删除的语法，在弹出菜单后点击卸载

## 支持语法
- Vue
- Lang
